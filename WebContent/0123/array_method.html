	slice(start, end) : start ~ end - 1까지의 요소를 짤라서 새로운 배열을 만든다.
	
	splic() : 배열요소를 삭제 / 변경
	var fruits = ["Banana", "Orange", "Apple", "Mange"];
	
	fruits.splice(2, 0, "Lemon", "Kiwi");
	---> 2번째 위치에서 0개를 지우고(지우지 않고) 2번째위치 부터 값을 삽입한다.
	
	fruits.splice(2, 1, "Lemon", "Kiwi");
	---> 2번째 위치에서 1개를 지우고(지우지 않고) 2번째위치 부터 값을 삽입한다.



	배열합치기 / 배열문자열로 변환하기

	a.join(b) : a배열과 b배열이 하나의 문자열로 변환한다.
