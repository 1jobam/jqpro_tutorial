fieldset : 요소들의 그룹핑에 사용
	label : 화면에 표시되는 일반문자
	display : 화면에 표시되는 속성을 설정
				inline : 가로세로를 지정할수 없다.
				block : 가로세로 길이를 설정할수 있으며 자동으로 줄이 바뀐다.
				inline-block : 줄은 자동으로 바뀌지 않지만 가로세로를 지정할수 있다.
				none : 화면상에 표시가 없도록 한다.
	<input type="range" : onmousemove="getValue()", onchange="getValue()">
